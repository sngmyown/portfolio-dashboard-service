Features

3계좌 통합 구조

보유 종목 관리

보유 종목 추가, 수정, 삭제와 원금·평가금액·평가손익·수익률 표시를 지원합니다.

환율 적용

USD 자산은 달러로 입력하고, 설정한 USD/KRW 환율을 적용해 원화 평가금액으로 계산합니다.

분할 매수·매도 기록

날짜별 분할 매수, 분할 매도, 적립식 매수, 추가 납입 기록을 관리할 수 있습니다.

스윙매매 복기

장기투자 목표 비중

자산군별 목표 비중을 설정하고 현재 비중과 비교할 수 있습니다.

배당·커버드콜 현금흐름

FAQ

자주 묻는 질문

증권사 계정 정보를 입력해야 하나요?

아닙니다. 증권사 로그인 정보, 비밀번호, API 키를 요청하지 않습니다.

데이터는 어디에 저장되나요?

기본 구조는 브라우저 내부 저장소를 사용합니다. 고객은 직접 백업 파일을 저장하고 필요할 때 복원할 수 있습니다.

투자 조언도 포함되나요?

아닙니다. 본 서비스는 투자 데이터 정리 및 시각화 대시보드 제작 서비스입니다.

나중에 종목이나 기록을 직접 수정할 수 있나요?

가능합니다. 보유 종목, 스윙 기록, 배당/커버드콜 기록, 장기투자 목표 비중은 직접 추가/수정/삭제할 수 있습니다.

모바일에서도 사용할 수 있나요?

반응형 화면을 지원하지만, 표와 입력 항목이 많기 때문에 PC 또는 태블릿 사용을 권장합니다.
